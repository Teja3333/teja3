HTML (Hypertext Markup Language) is the standard markup language used to create web pages. Here's a brief overview:

What is HTML?

HTML is a set of codes (tags) used to structure and organize content on the web.

Basic HTML Structure:

1. !DOCTYPE html : Declaration
2. : Root element
3. : Contains metadata (title, styles, scripts)
4. : Contains content (text, images, links, etc.)

Common HTML Elements:

1. Headings (h1,h2, h3, etc.)
2. Paragraphs p>
3. Links (a)
4. Images (img)
5. Lists (ul, ol, li)
6. Containers (div, span)

HTML Attributes:

Additional information added to tags, such as: 1. href (link destination)
2. src (image source)
3. alt (image alternative text)
4. title (tooltip text)
HTML Versions: 1. HTML4 2. HTML5 (current version)
